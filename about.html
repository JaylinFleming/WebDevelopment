About Me

Hi, I’m Jaylin Fleming — an Information Technology major, minor in Software Engineering, with focus on Cybersecurity - I'm passionate about web development, user experience, and emerging technologies. I love creating sleek, responsive interfaces and figuring out how software relates to the physical world.

I created this website as part of my IT 3203: Intro to Web Development project at Kennesaw State University. The topic I chose — the history and evolution of GPUs — perfectly matched my dual interest in hardware performance and digital design. The project helped me learn how to use HTML, CSS, and Javascript to build an informative and visually interactive webpage with multi-page, responsive features.

In this course and beyond, I hope to continue to expand my technical skills with web systems and security as well as apply those skills to other design projects in where performance, usability, and creativity clash.
